## Installation

##### Git:

```
git clone https://github.com/aslanon/vue-formulize.git
```

##### NPM: 

```
npm install --save vue-formulize
```

```
import VueFormulize from "vue-formulize"
```

```
...
components: {
  VueFormulize
}
...

<template>
...
  <vue-formulize></vue-formulize>
...
</>
```
